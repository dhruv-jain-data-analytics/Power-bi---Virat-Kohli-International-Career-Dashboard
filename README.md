
# Virat Kohli Internationsl Career Dashboard (Power BI)

## Project Overview

This Power BI dashboard provides a comprehensive analysis of Virat Kohli's international cricket career. The project explores his batting performance, format-wise dominance, yearly run trends, consistency, leadership impact, and captaincy record using interactive visualizations.

## Business Objective

The objective of this project is to transform cricket statistics into meaningful insights through data visualization and analytics. The dashboard helps analyze performance trends, leadership effectiveness, and career milestones across different formats of the game.

## Key Insights

### Career Performance

* Virat Kohli has accumulated more than 28,000 international runs across all formats.
* He has registered 85 international centuries and 146 half-centuries.
* Peak run-scoring years were observed during the 2016–2019 period.

### Format Dominance

* ODI cricket contributed the highest share of total runs.
* ODI format also recorded the highest batting average.
* Kohli maintained strong performance across all three international formats.

### Century Analysis

* The highest number of centuries were recorded during 2017 and 2018.
* Consistent century production highlights his dominance during the peak phase of his career.

### Batting Performance

* Performance while chasing was significantly stronger, showcasing exceptional chase-master abilities.
* Strong consistency was maintained regardless of match situation.

### Leadership Analysis

* Captained India in 213 international matches.
* Led India to 135 victories as captain.
* Achieved an overall captaincy win percentage of approximately 64%.

### Captain vs Non-Captain Comparison

* Maintained excellent performance both as captain and as a player.
* Demonstrated the ability to balance leadership responsibilities with batting excellence.
